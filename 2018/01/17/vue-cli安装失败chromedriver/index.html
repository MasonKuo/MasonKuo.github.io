vue-cli安装失败chromedriver

发表于 2018-01-17

在安装vue-cli脚手架时遇到了错误
vue-cli failed at the chromedriver@2.34.1 install script

要解决这个问题很简单，只需要安装cnpm 并在vue init最后一步选择自定义安装就行了
首先安装cnpm
npm install -g cnpm –registry=https://registry.npm.taobao.org
接着执行vue init ….
最后选择安装方式
No, I will handle that myself
然后进入刚才init的文件夹。
执行 cnpm install，等待安装完成
执行cnpm run dev
